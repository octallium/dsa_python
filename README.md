# Data Structures & Algorithms in Python

Collection of various Data Structures and Algorithms in Python

## Setup

1. Make sure [Poetry](https://python-poetry.org/) is installed
2. Clone Repository - 
   
   ```bash
   git clone github.com/octallium/dsa_python
   ```
3. ```bash
   cd dsa_python
   ```
4. ```bash
   cd poetry install
   ```
5. ```bash
   cd poetry shell
   ```
6. Testing
   
   ```bash
   pytest -vv
   ```
